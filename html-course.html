HTML Basics

What HTML is, tags, elements, and attributes. HTML documents are text files with elements enclosed in < >.

Document Structure & Head

Important head elements: <meta charset>, viewport, title, links to CSS, and scripts. Use semantic <main>, <header>, <footer> for layout.

Text, Headings & Paragraphs

Headings (<h1>–<h6>), paragraph <p>, <br>, <small>, emphasis <strong> <em>.

Links & Images

Anchor tags <a href> (internal & external), target="_blank", rel="noopener". Images with alt attribute for accessibility.

Lists

Ordered <ol>, unordered <ul>, definition <dl> lists; nesting, accessibility roles.

Tables

Use <table>, <thead>, <tbody>, <tr>, <th>, <td>. For accessibility add scope attributes and captions.

Forms & Inputs

Form elements: <form>, <input>, <select>, <textarea>, <button>, validation, labels, fieldset, legend, and accessibility concerns.

Notes: Always label inputs with <label>, use appropriate input types, and use client + server validation. For file uploads use <input type="file"> and for dates use <input type="date">.

Audio & Video

Use <audio controls> and <video controls>. Provide captions and fallbacks for accessibility.

Semantic HTML

Semantic elements (<article>, <section>, <nav>, <aside>, <header>, <footer>) improve structure & accessibility and help SEO.

Accessibility (a11y)

Use alt attributes for images, ARIA roles when appropriate, keyboard focusable elements, labels for inputs, and semantic markup. Test with a screen reader and keyboard only.

SVG & Canvas

SVG is XML for vector graphics — great for icons & scalable shapes. Canvas is a bitmap drawing surface for dynamic graphics.

Meta tags & SEO basics

Important meta tags: charset, viewport, description, open graph & Twitter cards. Use semantic tags & accessible content. Provide meaningful titles and descriptions for better indexing.

Advanced: Templates & Web Components

HTML templates & Web Components enable reusable UI. Template tag keeps HTML inert until you clone it. Web Components include custom elements, shadow DOM, and HTML templates.
